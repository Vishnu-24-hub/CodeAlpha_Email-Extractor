# CodeAlpha_Email-Extractor
# 📧 Email Extractor using Python

## 📌 Project Overview

This project was developed as part of the **CodeAlpha Python Programming Internship**.

The Email Extractor is a Python application that scans a text file, identifies valid email addresses using **Regular Expressions (Regex)**, removes duplicate entries, and stores the extracted email addresses in a separate output file.

---

## 🚀 Features

- 📂 Reads text from an input file
- 📧 Extracts valid email addresses using Regular Expressions (Regex)
- 🔄 Removes duplicate email addresses
- 💾 Saves the extracted emails to an output file
- 📊 Displays the total number of unique email addresses found

---

## 🛠️ Technologies Used

- Python 3
- Regular Expressions (`re` module)
- File Handling
